# **Ngab Trip** 
## _Aplikasi Rombongan Perjalanan Wisata_

Ngab Trip adalah aplikasi yang membantu para pengendara motor/Ngabers membentuk rombongan (BUKAN CIRCLE!!!) untuk berbagai destinasi wisata.
Pengguna dapat membuat grup atau bergabung ke grup lain, jika membuat bisa menentukan meeting point, mengatur kapasitas anggota, hingga memberikan komentar setelah trip selesai.

Aplikasi ini dibangun menggunakan Kotlin + Jetpack Compose, serta Supabase sebagai backend database real-time menggunakan Supabase.
